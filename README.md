这是一个密码算法工具箱软件，包含大多数密码键盘的算法，您可以利用他做加解密、校验或者其他功能。本工具箱提供了丰富的密码算法支持，适用于多种应用场景。

## 功能特点

1. **算法支持**：
   - 对称密钥算法：支持常见的对称加密算法。
      - MAC算法：支持多种消息认证码算法。
         - PINBLOCK算法：支持PINBLOCK的生成和处理。
            - Hash算法：支持多种哈希算法。
